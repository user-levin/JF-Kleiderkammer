# Digitale Kleiderkammer – Jugendfeuerwehr (MVP)

Ziel: Bestand verwalten, Kindern zuordnen, Verlauf protokollieren, Scan-Workflow am Handy.
Kernbereiche: Bestand | Kinder | Verlauf | Scan. Helme mit Ablaufdatum.

## Setup (Backend, Platzhalter)
- Python 3.11+
- `pip install -r requirements.txt`
- `.env` aus `.env.example` kopieren und Werte setzen.

## Ordner
- app/         → später Backend-Code (API)
- scripts/     → CSV Import/Export
- tests/       → Tests
